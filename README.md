# 배고팟

## ✏ 프로젝트 개요
- 혼밥 하기 싫은 사람들을 위해 '파티매칭' 기능을 더해 주변식당들의 모든 정보를 담은 홈페이지
- 당시 학원 수강중에 배운 기술들을 모두 접목시켜 최대한 많은 기능을 구현하기 위한 프로젝트.
- 외부 api (네이버지도 등)을 이용해보았고, 웹사이트의 기본 구조들을 이해하기 위함.
